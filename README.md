# Veritas

Veritas is an AI-driven platform that transforms scattered citizen complaints into structured insights and ensures real-world action through intelligent prioritization, worker assignment, and accountability.

---

##  Overview

Local communities generate a huge amount of data through complaints, surveys, and field reports. However, this data is often unstructured and scattered.

**Veritas solves this by:**

* Converting raw inputs (images, videos, text) into meaningful insights
* Identifying urgent issues using AI
* Connecting problems to the right workers
* Ensuring accountability through tracking and escalation

---

##  Key Features

###  Citizen Portal

* Secure login (DigiLocker-style simulation)
* File complaints:

  * AI Mode (upload image/video → auto-generated report)
  * Manual Mode (form-based)
* AI-generated:

  * Category detection
  * Severity & priority
  * Complaint summary
* Track complaints via unique Track ID
* View submission history
* AI-based social escalation if unresolved

---

###  Worker Portal

* Worker login with unique ID
* Dashboard showing:

  * Assigned tasks
  * Priority levels
  * Map view (location-based tasks)
* Task details with citizen info and severity
* Department-based task allocation

---

###  Admin / Control Office

* AI-powered dashboard:

  * Total complaints
  * Solved vs pending
  * Urgent alerts
* Complaint management system:

  * All / Urgent / Fake complaints
* AI features:

  * Fake complaint detection
  * Auto summarization
  * Priority assignment
* Worker assignment:

  * AI-based or manual
* Status tracking:

  * Under Process / Solved / Pending
* AI-generated rejection messages (SMS)

---

##  Demo Flow

1. Citizen uploads a complaint (image/video)
2. AI generates structured report
3. Admin reviews and approves
4. AI assigns worker automatically
5. Worker receives task with location
6. Delay triggers AI-based social escalation
